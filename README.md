Library Management System (DBA 120 Final Project)

Introduction

This is a console-based Library Management System created for SQL Database Programming I (DBA 120) at Sandhills Community College. This project represents relational database structures, raw SQL development and implementation, and application-level integration using Python/MySQL.

The application itself will manage books, patrons, staff, checkouts, returns, reservations and reporting analytics for a public library setting.

---

Technical Specifications

- Language Used: Python
- Database Used: MySQL
- Driver: mysql-connector-python
- Library Used: Console based application
- SQL Used: Raw SQL only - no ORM taken advantage of here.

---

Functionalities

- Staff login validation
- Book management (add, update, delete)
- Patron management (add, update, delete)
- Check out and return books
- Reserve books
- Check inventory (available/checked out)
- Reporting:
- Most popular books
- Overdue books
- Patron statistics
- Available checked out statistics
- Monthly/Yearly reports on circulation
